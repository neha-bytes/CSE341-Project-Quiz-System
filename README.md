# CSE341-Project-Quiz-System
QUIZ SYSTEM 


Feature 1: Powerskip; Used only once
A power move alongside each question that can be used only once in every game, which will skip that question without losing points.

Feature 2: Choose Quiz Categories
Allows users to select categories (e.g., Cars, Aeroplanes, Motorbikes).
Filters questions based on the selected category before starting the quiz.
Enhances user experience by making the quiz more personalized.

Feature 3: Show Questions One by One & 30sec Time Limit
Participants will have only 30 seconds per question, and questions will be presented sequentially during the quiz. 
Ensures a smooth and controlled quiz flow.
Prevents skipping ahead unless designed otherwise.

Feature 4: Take User Answers & Calculate Score
Accepts and validates user input for each question (A, B, C, D).
Compares responses to correct answers and dynamically tracks the score.
Maintains a running total of correct answers throughout the quiz.

Feature 5: Show Correct Answers at End
Displays correct answers after quiz completion.
Provides a summary of user performance and mistakes.
Helps users learn from incorrect responses.

Feature 6: Store High Scores
Maintains the highest score achieved using array storage.
Updates the record when a new high score is reached.
Allows basic performance tracking across multiple attempts.
